## 原理
+ 使用静态页面进行内容编排
+ 使用运行在客户端的`JavaScript`控制页面，进行信息传递
+ 利用`createObjectURL`传递生成本地图片地址
+ 本项目为了增加安全性，未将任何用户信息上传至服务端，所有人可放心食用
